## Notes for Designers

1. **Modal Scrolling:** For long candidate lists, make modal scrollable internally (not full page)
2. **Touch Targets:** Minimum 44px height for buttons on mobile
3. **Tab Indicator:** Use both color + underline (not color alone)
4. **Placeholder Text:** Use gray (#999) at 14px weight
5. **Form Labels:** Left-aligned, bold, 12px uppercase
6. **Action Buttons:** Primary (blue), Secondary (gray), Danger (red)
